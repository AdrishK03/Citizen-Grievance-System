# 🏛️ AI-Driven Citizen Grievance & Sentiment Analysis System

## 📌 Overview
This project is an AI-powered system that automates the classification and prioritization of citizen complaints using Natural Language Processing (NLP).

The system:
- Classifies complaints into departments (Electricity, Sanitation, etc.)
- Detects sentiment (Normal, Negative, Critical)
- Assigns a priority score for faster resolution

---

## 🚀 Features
- ✅ Complaint classification using SVM
- ✅ Sentiment analysis using VADER
- ✅ Real-time API using FastAPI
- ✅ Interactive UI using Streamlit
- ✅ Priority-based issue ranking

---

## 🧠 Tech Stack
- Python
- Scikit-learn (SVM, TF-IDF)
- NLTK (Preprocessing + Sentiment)
- FastAPI (Backend API)
- Streamlit (Frontend UI)
